【メッセージ枠調整プラグイン Ver.2.00】
http://hororo.wp.xdomain.jp/56/

2017/6/8更新 v454対応版

-------------------------------------------------------------------------------------------------
■機能
-------------------------------------------------------------------------------------------------
　全てのメッセージエリアで、縁取り・影設定を有効にするプラグインです。
　
　◆できる事
　　・縁取り・影を同時に両方付けられます。
　　・キャラ名にも縁取り・影を付けます。
　　・縦書き時も有効です。

　◆できない事
　　・ゲームの途中で色やサイズ変更はできません。
　　・特定のエリアのみを指定したりはできません。
　　・あんまり細かい指定はできません。
　　・FONTの種類によっては綺麗に縁取り出来ない場合があります。

-------------------------------------------------------------------------------------------------
■使い方
-------------------------------------------------------------------------------------------------
　①data/others/plugin に 「message_edge」を入れてください。
　
　②first.ks からプラグインを呼び出してください。
　　[plugin name="message_edge"]

　◆パラメーター（※デフォルトの場合は省略可）
　　・縁取り（8方向）
　　　edge　　　　＝　"true"　…　※省略時はdefaultEdgeを参照します。
　　　edge_color　＝　"Ox000000"　…　色 0xRRGGBB 形式※省略時はdefaultEdgeColoになります。
　　　edge_blur　 ＝　"0"　…　ぼかす数値　※省略時は0
　　　edge_offset ＝　"1"　…　縁のサイズ　※省略時は1 大きくし過ぎるとアレです。
　　　　　
　　・影（右下固定）
　　　shadow　　　　＝　"true"　…　※省略時はdefaultShadowを参照します。
　　　shadow_color　＝　"Ox000000"　…　色 0xRRGGBB 形式※省略時はdefaultShadowColorになります。。
　　　shadow_blur　 ＝　"0"　…　ぼかす数値　※省略時は0
　　　shadow_offset ＝　"1"　…　縁のサイズ　※省略時は1


　◆記述例
　　例1：Config.tjsで true 指定してる場合はパラメータなしでもOK。
　　　　[plugin name="message_edge"]
　　例2：縁取りの色指定をする
　　　　[plugin name="message_edge" edge_color="0xFF0000"]
　　例3：影のぼかし指定をする。
　　　　[plugin name="message_edgr" shadow_blur="2"]
　　例4：全部指定する
　　　　[plugin name="message_edge" edge="true" edge_color="0xFF0000" edge_blur="2" edge_offset=2 shadow="true" shadow_color="0x00FF55" shadow_blur="1" shadow_offset="3"]

　◆余談
　　edgeは8方向指定にしているので、blurを大きくしても濃い色味を保ちます。
　　shadowは1方向のみなので、blurを大きくすると薄くなります。
　　細かく指定したい場合はCSSで直接指定した方が良いです。
